# C++ Keylogger

A simple keylogger written in modern C++ for educational and research purposes. This project demonstrates low-level keyboard input handling, event processing, and file logging on supported platforms.

## Features

- Keyboard event capture
- Log keystrokes to a file
- Lightweight and minimal implementation
- Written in modern C++
